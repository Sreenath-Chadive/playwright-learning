# Playwright TodoMVC (React)

E2E automation framework built with Playwright 

## Tech Stack
- Playwright Test
- TypeScript
- Cross-browser testing (Chromium)

## Project Structure
- `tests/` → test cases grouped by intent
- `pages/` → page objects (user behavior)
- `fixtures/` → test setup & context
- `helpers/` → shared utilities
- `test-data/` → static data

## Commands
```bash
npm test
npm run test:ui
npm run test:smoke
npm run test:report
